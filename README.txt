CHONLASIT NILSIRI — ORGANIZED PORTFOLIO
=======================================

ไฟล์เปิดเว็บ: index.html

โครงสร้างผลงานใหม่
-------------------
projects/project.html       หน้าแสดงรายละเอียดผลงานเพียงหน้าเดียว
projects/projects.js        ข้อมูลผลงานทั้งหมด แก้หรือเพิ่มที่ไฟล์นี้ไฟล์เดียว
projects/project.css        รูปแบบหน้ารายละเอียดผลงาน
projects/project.js         ระบบแสดงข้อมูลและเอฟเฟกต์ของหน้าผลงาน
projects/HOW-TO-ADD-PROJECT.txt  คู่มือเพิ่มผลงาน

ระบบนี้ไม่ต้องสร้าง HTML ใหม่แยกทุกผลงานอีกต่อไป
หน้าเว็บหลักและดีไซน์ของ Chonlasit ยังคงเดิม โดยการ์ดแต่ละงานจะเปิดหน้า project.html พร้อม id ของผลงาน

เวอร์ชัน Mobile + Video Preview
- ช่องผลงานแสดงทันทีบนมือถือ ไม่พึ่งเอฟเฟกต์เลื่อนหน้าจอ
- ปุ่ม “เล่นตัวอย่าง” เปิดวิดีโอ YouTube / Google Drive / Instagram ในเว็บ
- วิดีโอ Facebook จะเปิดบนแพลตฟอร์มต้นทาง
- ผลงานที่ยังไม่มี URL จะแสดงข้อความว่า “ยังไม่มีวิดีโอ”
- เปิดใช้งานโดยดับเบิลคลิก index.html
